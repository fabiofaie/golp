# Icone ambiente DEV (verde)

Inserire qui le icone PWA colorate in verde (`#22C55E`) nei formati:

| File | Dimensione |
|---|---|
| `icon-72x72.png` | 72×72 px |
| `icon-96x96.png` | 96×96 px |
| `icon-128x128.png` | 128×128 px |
| `icon-144x144.png` | 144×144 px |
| `icon-152x152.png` | 152×152 px |
| `icon-192x192.png` | 192×192 px |
| `icon-384x384.png` | 384×384 px |
| `icon-512x512.png` | 512×512 px |

Finché le icone non esistono, la PWA installata su dev mostrerà icone mancanti (HTTP 404),
ma l'app funziona normalmente e il logo nell'header è già verde.
