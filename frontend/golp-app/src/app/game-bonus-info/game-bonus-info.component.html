Come funziona il punteggio

Il metodo Game+Bonus: punti dai game giocati, senza rating assoluto.

ELO
⚖️ ELO vs Game+Bonus: le differenze
ELO è un rating assoluto e persistente: sale o scende ad ogni partita confermata e resta valido finché non giochi ancora. Game+Bonus invece somma punti guadagnati dai game giocati (più un bonus se batti chi è più forte) solo sulle partite recenti: una partita vecchia esce dal conteggio da sola, senza rating da "portarsi dietro". Il proprietario del circolo sceglie quale metodo è attivo, ed è reversibile in ogni momento senza toccare le partite già giocate.
Le basi
🎾 Punteggio base
La squadra vincente guadagna 1 punto per ogni set vinto (meno 1 per ogni set perso), più la differenza di game nei soli set che ha vinto. I set persi non tolgono punti oltre a questo: vincere la partita 2 set a 1 paga sempre, anche se in un set hai perso largo. La squadra perdente non guadagna punti in quella partita.
🏆 Bonus se batti i più forti
Se la tua squadra aveva un punteggio Game+Bonus inferiore a quello degli avversari prima della partita, vincendo ricevi un bonus aggiuntivo pari al 10% della differenza di punteggio tra le due squadre.
🗓️ Solo le partite recenti contano
La classifica del circolo somma solo i punti delle partite che rientrano sia nelle ultime N partite del circolo sia nelle ultime M settimane (parametri impostati dal proprietario). Una partita che esce da una delle due finestre smette di contare.
