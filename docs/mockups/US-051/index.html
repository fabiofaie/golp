Circolo Padel Milano — Impostazioni

Metodo di calcolo punteggio
Rating ELO

Il metodo attuale. Ogni vittoria/sconfitta sposta il rating in base alla differenza di forza tra le squadre. Storico completo, nessuna finestra temporale.

Esempio: vinci 6-4 contro una squadra più forte → +18 pt rating
Game + Bonus

Punti = differenza game + 1 per la vittoria. Bonus se vince la coppia più debole. Conta solo il periodo recente: le partite più vecchie escono dal conteggio.

Esempio: vinci 6-4 → 2 + 1 = 3 pt. Se eri sfavorito, +10% del divario in classifica
Rolling: entra la nuova, esce la più vecchia
Partite più vecchie non contano più
✓ Salvato
