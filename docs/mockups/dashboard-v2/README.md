# Dashboard v2 — proposta di revisione

La schermata trasforma la dashboard da menu di link a centro operativo personale.

- Prima ciò che richiede un'azione: conferme e contestazioni.
- Poi il contesto attivo: circolo, rating, posizione e andamento.
- Quindi le ultime partite, in forma compatta e leggibile.
- Navigazione primaria persistente in basso; “Registra partita” resta sempre raggiungibile.
- Un solo colore primario per le azioni; verde e rosso descrivono esclusivamente gli esiti.

Il mockup è intenzionalmente mobile-first e usa dati già presenti nel dominio GOLP. Per implementarlo servirebbe un endpoint aggregato (o più chiamate coordinate) per riepilogo rating, posizione e statistiche del circolo attivo.
